# Integrations

Nagi integrates with other software through [Origin](../reference/resources/origin.md) to automatically generate resources such as Asset and Conditions.

| Integration | Description |
| --- | --- |
| [dbt Core](./dbt/core.md) | Automatically generate resources from a dbt Core project |
| [dbt Cloud](./dbt/cloud.md) | Operational model and configuration for dbt Cloud environments |
